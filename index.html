Map-Wall Live January 14th, 2017 at the California Science Center in Los Angeles!

We went live January 14th, 2017! Check out the exhibit I did all the coding for at the California Science Center in Los Angeles. Map-Wall is simple to use and requires users to point the spotlight at one of the several points of interest on the map and press the trigger, which causes information on the monitor to be displayed about the selected point.
This projects backend was written in all C++ that leveraged ultraDMX Micro for interfacing with DMX512, SEI Absoluter Encoder Comm Protocol for reading (X,Y) coordinates, and U-HID for the LED Push Buttons on the spotlight device.
The frontend parses JSON generated by an Excel file filled out by the exhibit currator which the code supports English and Spanish which users can change with a button press on the base of the device. The frontend is generated on the fly using basic HTML,CSS, and Javascript leveraging the dojo toolkit. All points are pre-calibrated for each station with extra space for more in the future.
This has been one of my favorite projects I have done thus far. It is is awesome to see people of all ages interacting with and enjoying the exhibit!
